SI 1 - DESIGN DE SOFTWARE


Sistemas de Informação 1, uma disciplina que pode ser considerada um divisor de águas quando o assunto é programar


A diferença entre saber programar e fazer um software Profissional é imensa, é preciso conhecer as principais metodologias e além disso saber aplicá-las para colocar um software no mercado.SI 1 como disciplina visa favorecer isso.
