SI 1 - Design de Software

Sistemas de Informação 1, uma disciplina que pode ser considerada um divisor de águas quando o assunto é programar.

Programa X Software

A diferença entre saber programar e fazer um software Profissional é imensa, é preciso conhecer as principais metodologias e além disso saber aplicá-las para colocar um software no mercado.SI 1 como disciplina visa favorecer isso.

Metodologia

As metodologias e práticas oferecidas em SI 1, tornam um programador capaz de conceber um software de maneira mais concisa, reduzindo as abstrações e tornando o seu software ou seu projeto em algo que de fato será "usável" do ponto de vista comercial, ou seja, daquilo que se espera de um software.

Oportunidades

Ao fim da disciplina, tendo aprendido e seguido as metodologias de maneira correta e através das diversas práticas, você estará apto a conceber um software de maneira completa. Para isto, clique no link abaixo e se matricule, não perca esta chance nao adie mais nenhum período!
